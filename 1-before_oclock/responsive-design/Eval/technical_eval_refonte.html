Introduction

Ruby est un langage de programation générique et open-source, orienté objet.
Sa conception (alpha en 1994) a été inspirée par les langages Smalltalk, Eiffel,Ada et Lisp ainsi que le Perl dont il emprunte la bibliothèque de classe.


Installer Ruby

Afin d'installer ruby sous Archlinux, vous pouvez passer par votre gestionnaire paquets.

$ sudo pacman -S ruby

Vous pouvez également le construire depuis la source.
Après avoir téléchargé et extrait le code source, lancez:

$ ./configure
$ make
$sudo make install

Pour l'installer sous Windows, un projet intéressant est le RubyInstaller qui installera les prérequis ainsi qu'un environnement de développement.


Interactive Ruby

Interactive Ruby, IRB en court, permet d'exectuer des programmes en ruby depuis la console. L'un des premiers exemples est la calculatrice.

on commence par appeler IRB.

user@pc:~$ irb

Puis on peut calculer.

irb(main):> 2+5
=> 7

De cette façon, IRB permet de tester son code assez rapidement. Cependant, par défaut IRB ne conserve pas l'historique des commandes, empechant ainsi l'execution de codes à variables.


Les types d'objets en Ruby

Chaque objet que vous utilisez en Ruby est un literal.
Les literal créent les objets utilisés dans les programmes.

    Les types de literals en Ruby

  • Booléens
  • Nombres
  • chaines de caractères
  • symboles
  • Matrices
  • Hashes
  • Etendues
  • RegEx
  • Processus

Changer pour Ruby

Si vous connaissez un autre langage (C/C++, JAVA,PERL, PHP ou encore Python), VOus pouvez aisément passer à ruby.
Voici un exemple.

# en Python
if 0:
print("0 est vrai/true")
else:
print("0 est faux/false")

# en Ruby
if 0
puts "0 est vrai/true"
else
puts "0 est faux/false"
end
