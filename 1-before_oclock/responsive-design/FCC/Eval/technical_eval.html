Introduction

At its heart, HTML is a language made up of elements, which can be applied to pieces of text to give them different meaning in a document,structure a document into logical sections,and embed content such as images and videos into a page.

This module will introduce the first two of these and introduce fundamental concepts and syntax you need to know to understand HTML.

Prerequisite

ou don't need any previous HTML knowledge, but you should have at least basic familiarity with using computers and using the web passively.

You should have a basic work environment set up, and understand how to create and manage files.

head

The head of an HTML document is the part that is not displayed in the web browser when the page is loaded.

This page's head looks like this

<head>

<meta name='viewport' content='width=device-width,initial-scale=1.0'/>

<meta charset='utf-8'/>

<link rel='stylesheet' type='text/css' href='./styles.css'/>

<Title>HTML Technical documentation/title </head>"

main

The <main> tag specifies the main content of a document.

he content inside the <main> element should be unique to the document. It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search forms.

Mise en forme

Afin de donner son aspect visuel à une page HTML>/strong>, un deuxième fichier "nom.css" est souvent utilisé.

Dans celui-ci, de nombreux éléments peuvent être définis

  • les blocs de fonction comme <body>

    body { min-width: 290px; color: #4d4e53; background-color: #ffffff; font-family: 'Open Sans', Arial, sans-serif; line-height: 1.5; }

  • les classes à l'interieur des balises #navbar a { display: block; padding: 10px 30px; color: #4d4e53; text-decoration: none; cursor: pointer; }

  • Les aspects réactifs au média utilisé pour l'accès

    @media only screen and (max-width: 815px) { /* For mobile phones: */ #navbar ul { border: 1px solid; height: 207px; }

  • Les ID des differentes balises utilisées

    .keys { background-color: #040404; width: 949px; height: 180px; padding-left: 2px; overflow: hidden; }

  • Mais aussi les apparences en fonction des actions(sur les liens par ex.)

    a:visited { color: black; } a:hover { color: brown; } a:active { color: brown; }
